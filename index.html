Serving Static Files With NodeJS

Install NodeJS, and install via NPM 'http-server'. Forward port 80 on your router, and if you have a domain name, make sure it is mapped correctly to your server's IP. Also, to run http-server in the background, install tmux.
Open xterm or any shell window on your server and type:

$ cd folder
to change your current directory to the folder of the website.
Next, type:
$ tmux
to open a new terminal window.
Next, type:
$ sudo http-server -a address -p port
to run the http-server.
Finally, hit ctrl-b, d to background and detach tmux.
To shut down the server:
$ tmux a
and type 'exit'.
